# ジムコミ(スポーツジムの口コミサイト)
# 概要
## サイト内容
- スポーツ施設（フィットネスジム）の口コミサイト
- 当該施設を施設を利用した人からの口コミ情報を収集するサイト
## 対象
- 近所に複数のジムがありどこのジムを利用するか悩んでいる人
- ジムをこれから利用したいけど、ジムの雰囲気を事前に知りたいと考えている人
